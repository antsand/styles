üìÇ Accordion

Accordion Component
Collapsible content panels (auto-initialized with JavaScript)

ANTSAND Styles v2 is a modern CSS library with a Bootstrap-like philosophy, providing semantic classes for consistent design.

Unlike Tailwind's utility-first approach, ANTSAND uses semantic classes that describe purpose, not pixels. This creates more consistent and maintainable code.

Interactive components like accordion, tabs, and modal require the included JavaScript file. Include antsand.js and components auto-initialize.

<div class="antsand-accordion"> <div class="antsand-accordion-item"> <button class="antsand-accordion-header"> <span>Question</span> <span class="antsand-accordion-icon"></span> </button> <div class="antsand-accordion-body"> <div class="antsand-accordion-content">Answer</div> </div> </div> </div>

‚ö° JavaScript Integration

Auto-Initialization
All interactive components work automatically when you include antsand.js
üí°
Quick Start:
  1. Include the CSS: <link rel="stylesheet" href="/builds/production/css/antsand-v2/antsand.css">
  2. Include the JS: <script src="/path/to/antsand.js"></script>
  3. Components auto-initialize on page load!

Manual Initialization (Optional)

// Initialize all components
Antsand.init();

// Or initialize specific components
Antsand.accordion.init();
Antsand.tabs.init();
Antsand.modal.init();
Antsand.navbar.init();
Antsand.dropdown.init();
Antsand.alert.init();

// With custom selectors
Antsand.accordion.init('.my-custom-accordion');

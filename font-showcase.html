Usage Guide

How to Use These Fonts

1. Include ANTSAND v2 Styles

<link rel="stylesheet" href="/builds/production/css/antsand-v2/antsand.css">

2. Apply Font Classes

<h1 class="font-Graphik">Headline</h1> <p class="font-Inter">Body text</p> <span class="font-GHGuardianHeadline">Editorial headline</span>

3. Combine with Size Utilities

<h1 class="font-Graphik font-size-48">Large Headline</h1> <p class="font-Inter font-size-16 line-height-1-6">Readable body text</p>

4. Responsive Typography

<h1 class="font-Graphik font-size-32 font-size-48-lg"> Responsive headline (32px on mobile, 48px on large screens) </h1>

5. Icon Usage

<i class="pictos-check"></i> <i class="font-Fa-paragraph"></i>

Font Hosting

All fonts are self-hosted on https://styles.antsand.com/builds/production/fonts/ for optimal performance and reliability.

Available Size Classes

Font sizes from 5px to 200px: .font-size-{5-200}

Line Height Classes

.line-height-1, .line-height-1-2, .line-height-1-4, .line-height-1-5, .line-height-1-6, .line-height-2

Integration with Databoard

These fonts are fully integrated with the ANTSAND databoard system. Use them in:

  • Template CSS configurations
  • Data mapping style overrides
  • Global common_styles
  • Section-specific styling
